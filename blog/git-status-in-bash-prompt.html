Show the git status of the current repo in your bash prompt by ReadEvalPrint on Oct 2, 2011

Make sure you have the current version of git installed. In your ~/.bash_profile add:

export GIT_PS1_SHOWDIRTYSTATE=1

Then add this somewhere where your PS1 is defined.

\$(__git_ps1 \"(%s)\")

Here is mine, and you can see the git status at the end.

PS1="\[\e[32m\][\u@\h]\[\e[0m\]\[\e[37m\][\t]\[\e[0m\]\n\[\e[36m\][\w]\[\e[0m\]\$(__git_ps1 \"(%s)\")>"
