Easy absolute uri with Django template tags

for this to work you'll need to add django.core.context_processors.request to TEMPLATE_CONTEXT_PROCESSORS. Read https://docs.djangoproject.com/en/dev/ref/templates/api/#django-core-context-processors-request. Then to your template add:

{{ request.build_absolute_uri|slice:":-1" }}{% url 'some-view' %}
